## 6. 技術架構與設定(整理自疑難排解)
- **設定檔**:`settings.json`(主設定)+ `models.json`(模型清單)。
  - ⚠️ 已知:若 `models.json` 沒有對應模型,系統會**改用清單第一個模型**執行(已排定改善計畫)。
- **BYOK(Bring Your Own Key)**:自帶金鑰串接,可接 **Pioneer 平台**、自訂 LLM。
- **可接的後端**:
  - GitHub Copilot CLI(需訂閱;沒訂閱會 `Access denied by policy`）
  - Pioneer 平台
  - Google **Antigravity**(OAuth,`antigravity-oauth-token` / 用 Google Cloud Project 登入)
  - Gemini / Google AI Pro($300 Cloud 抵免額)
  - Groq(`gpt-oss-120b`,Will 推薦)、Ollama
- **能力**:建 **LINE bot**、**網頁截圖**、跑 24 小時自動任務。
